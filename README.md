# Binary-Image-Watermarking-Embedding-and-Extraction-Framework
本研究面向二值图像在数字化办公中广泛应用所带来的安全挑战，针对其像素冗余度低、视觉失真敏感的技术难点，开展水印嵌入与提取算法的设计与分析研究。 在技术层面，构建了多维可翻转性评分模型，通过分析像素的平滑度、连通性与孤立点特征，筛选视觉影响最小的像素作为修改对象。采用块级统计特征调制策略，以块内黑像素数的奇偶性承载水印信息，实现盲提取。引入基于密钥的像素混洗机制，解决可翻转像素分布不均问题并增强安全性。系统采用Python与OpenCV实现模块化架构，支持扫描文本、签名、二维码等多种二值图像。 功能上，实现了高隐蔽性水印嵌入、盲提取、基础鲁棒性保障与密钥安全管理，并对嵌入容量与计算复杂度进行理论分析。
